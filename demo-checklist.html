📊 Table Chart Enhancements
✓ Feature What It Does Status
Number Formatting Format numbers per column — Indian (1,00,000), International (100,000), European (100.000), Adaptive, with configurable decimal places New
Conditional Formatting Color-code cells based on rules — e.g. highlight values > target in green, below threshold in red. Supports numeric and text conditions New
Search & Filter Instantly search across all columns with match counter. Press Escape to clear. Helps find specific records in large tables New
Freeze / Pin Column Pin the first column so it stays visible when scrolling horizontally through many columns New
Zebra Striping Alternating row colors for better readability — especially useful for wide tables with many columns New
Column Reorder Drag-and-drop to rearrange column order. Keyboard accessible. Saved with the chart configuration New
Column Alignment Set each column to left, center, or right align. Auto-detection aligns numbers right and text left by default New
Color Themes Choose from preset table themes (Gray, Blue) that style headers, rows, borders, and zebra stripes consistently New
🔄 Pivot Table — New Chart Type
✓ Feature What It Does Status
Row & Column Dimensions Group data by multiple dimensions on rows and columns — e.g. rows by District, columns by Year. Like Excel pivot tables but connected to live data New
Drill-Down Click a row to expand into sub-groups — e.g. State → District → Block. Multiple levels of nesting supported New
Time Grain on Columns Choose time granularity per column dimension — view by Year, Quarter, Month, Week, or Day New
Subtotals & Grand Total Toggle row subtotals, column subtotals, and grand total. Automatically calculated for all metric columns New
Number Formatting Same per-metric formatting as table charts — Indian, International, European formats with decimal control New
Conditional Formatting Color-code metric cells by rules. Can scope rules to specific drill-down levels — e.g. different colors at State vs District level New
Nested Column Headers Multi-level column headers when pivoting by multiple column dimensions — clear visual hierarchy New
Pagination Server-side pagination at the top-level dimension group. Handles large datasets without freezing the browser New
Zebra & Freeze Column Same readability features as table charts — alternating row colors and pinned first column New
Color Themes Preset themes with coordinated header, row, border, subtotal, and grand total colors New

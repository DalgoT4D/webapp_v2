# Iconography

Dalgo uses **`lucide-react`** for all icons (imported in 185 files). Default size
`size-4` (16px) inside buttons; `w-6 h-6` for feature cards. Color follows the
text color (`currentColor`) — don't hardcode icon colors.

The set below is the **most-used icons in the app** (by import frequency), so reach
for one of these before adding a new glyph.

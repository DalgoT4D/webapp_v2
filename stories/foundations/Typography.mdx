# Typography

Font: **Anek Latin** (loaded via `next/font`, exposed as `var(--font-anek-latin)`).
Mono: `var(--font-anek-mono)`. Never set `font-family` per component.

The scale below mirrors the type ramp documented in `.claude/rules/components.md`.

<div style={{ display: 'flex', flexDirection: 'column', gap: 20 }}>
  <div>
    <div className="text-3xl font-bold">Page heading</div>
    <code style={{ color: 'var(--muted-foreground)' }}>text-3xl font-bold</code>
  </div>
  <div>
    <div className="text-xl font-semibold">Section / card / modal title</div>
    <code style={{ color: 'var(--muted-foreground)' }}>text-xl font-semibold</code>

## Weights

Anek Latin is a variable font. Use Tailwind weight utilities — `font-bold` for
headings, `font-semibold` for titles, `font-medium` for emphasis, `font-normal`
for body.

<div style={{ display: 'flex', flexDirection: 'column', gap: 8 }}>
  {[
    ['font-normal', 400],
    ['font-medium', 500],
    ['font-semibold', 600],
    ['font-bold', 700],
  ].map(([cls, w]) => (
    <div key={cls} style={{ fontWeight: w, fontSize: 18 }}>
      Dalgo data platform — <code style={{ fontSize: 12, fontWeight: 400 }}>{cls} ({w})</code>
    </div>
  ))}
</div>

## Line height

Body uses `leading-normal`; tight UI rows use `leading-tight`. Two custom overrides
exist in `globals.css` (`line-height: 1.25rem` and `1rem`) for compact tables.

<div style={{ display: 'flex', gap: 32, marginTop: 12 }}>
  <p style={{ lineHeight: 1.25, maxWidth: 220, fontSize: 14 }}>
    leading-tight — used in dense table cells and compact lists where vertical space is tight.
  </p>
  <p style={{ lineHeight: 1.6, maxWidth: 220, fontSize: 14 }}>
    leading-normal — used in body copy, descriptions, and help text for comfortable reading.
  </p>
</div>

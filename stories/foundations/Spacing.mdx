# Spacing

Tailwind v4 spacing scale (base unit `0.25rem` = 4px). Use these for padding,
margin, and `gap` — never arbitrary pixel values like `gap-[13px]`.

<div style={{ display: 'flex', flexDirection: 'column', gap: 8, marginTop: 16 }}>
  {[
    ['0.5', 2],
    ['1', 4],
    ['2', 8],
    ['3', 12],
    ['4', 16],
    ['5', 20],
    ['6', 24],
    ['8', 32],
    ['10', 40],
    ['12', 48],
    ['16', 64],
  ].map(([token, px]) => (
    <div key={token} style={{ display: 'flex', alignItems: 'center', gap: 16 }}>
      <code style={{ width: 64, fontSize: 12 }}>p-{token}</code>
      <div style={{ width: px, height: 16, background: 'var(--primary)', borderRadius: 2 }} />
      <span style={{ fontSize: 12, color: 'var(--muted-foreground)' }}>
        {px}px · {(px / 16).toFixed(3).replace(/0+$/, '').replace(/\.$/, '')}rem
      </span>
    </div>
  ))}
</div>

## Common usage in Dalgo

- Card padding: `p-4` / `p-6`
- Grid gaps: `gap-3` (chart-type cards), `gap-4`
- Page header padding: `p-6 pb-0` (see `rules/components.md` page layout)

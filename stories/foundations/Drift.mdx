# Design Drift

Places where Dalgo's UI says the same thing in different ways. Each row is the
**same concept, multiple implementations** — the design team picks the canonical
one, then engineering collapses the rest. This page is generated from a code audit
of `components/**` + `app/**` (June 2026).

> Why it matters: today there is no single source for color/typography, so every
> new screen re-invents values. These are the concrete divergences, not theory.

---

## 2. Brand teal — hardcoded instead of `var(--primary)`

`--primary` is `#00897B`. The rule says never hardcode it. ~10 places do anyway:

  <Swatch name="#00897b (lowercase)" value="#00897b" />
  <Swatch name="#89C7C1 (lightened)" value="#89C7C1" />

Offenders: `pipeline/overview/pipeline-bar-chart.tsx:23`,
`transform/ui-transform/canvas/CanvasMessages.tsx` (×6),
`explore/StatisticsPane.tsx:249,472`, `ui/animated-background-simple.tsx`.
**Fix:** reference `var(--primary)` (or read the CSS var for canvas/ECharts).

## 3. Primary text — `text-gray-900` vs `text-foreground`

Headings/labels hardcode gray instead of the theme token. **~117 occurrences.**
The difference is invisible in light mode but **breaks dark mode** (gray-900 stays
near-black on a dark background). Flip the Theme toolbar to see it.

<div style={{ display: 'flex', gap: 32, margin: '16px 0' }}>
  <div>
    <div style={{ color: '#111827', fontWeight: 600 }}>text-gray-900 (hardcoded)</div>
    <code style={{ fontSize: 11 }}>#111827 — does not react to theme</code>
  </div>
  <div>
    <div style={{ color: 'var(--foreground)', fontWeight: 600 }}>text-foreground (token)</div>
    <code style={{ fontSize: 11 }}>var(--foreground) — flips in dark mode</code>
  </div>
</div>

Worst files: `app/charts/page.tsx` (×8), `pipeline/orchestrate/pipeline-list.tsx` (×4),
`ChartTypeSelector.tsx:58`, plus share/report pages. **Fix:** `text-foreground`
for primary text (secondary `text-gray-500` already maps fine → keep or move to
`text-muted-foreground`).

## How to fix the root (not just see it)

This page makes drift **visible**. To make it **impossible**:

1. One `tokens.json` (DTCG) → built by Terrazzo/Style Dictionary into both
   `globals.css` and a shared `palette.ts`.
2. `chart-types.ts` + `reports/utils.ts` + sidebar import from that palette.
3. Lint rule banning raw hex in `className`/`style` and bare `text-gray-*` for text.
4. One tooltip: drop the black overrides or add a real `variant`.

Storybook shows the gap; the pipeline + lint close it.
